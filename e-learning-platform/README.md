## E-Learn Pro – Modern E-Learning Platform (Frontend)

E-Learn Pro is a professional multi-page e-learning platform frontend built with **pure HTML5 and CSS3**.  
It showcases a modern UI inspired by real-world platforms like Coursera, Udemy, and Skillshare, including a marketing homepage, authentication flow, and a learner dashboard.

### Pages

- **`index.html`** – Homepage (hero, featured courses, features, categories, testimonials, CTA, footer)
- **`signup.html`** – Create account form
- **`signin.html`** – Login form (navigates to dashboard)
- **`forgot-password.html`** – Request password reset OTP
- **`verify-otp.html`** – OTP verification form
- **`reset-password.html`** – New password form
- **`dashboard.html`** – Learner dashboard with sidebar, cards, progress, and profile summary

### Technologies Used

- **HTML5** – Semantic multi-page layout and navigation
- **CSS3** – Modern styling with:
  - Flexbox and CSS Grid for layout
  - Responsive design breakpoints
  - Reusable components (navbar logo, buttons, cards, footer)
  - Hover animations and subtle shadows
- **Google Fonts** – `Poppins` for clean typography
